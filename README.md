# 🧹 CleanerBot — Telegram бот для организации уборки

**CleanerBot** — это Telegram-бот, созданный для координации уборки в общих помещениях (квартиры, офисы, общежития). Бот позволяет пользователям создавать комнаты, присоединяться к ним и управлять задачами по уборке (функционал задач в разработке).

---

## ✨ Особенности

- ✅ **Автоматическая регистрация** — при первом сообщении пользователь регистрируется в системе.
- ✅ **Создание комнат** — команда `/createroom <код>` создаёт новую комнату с уникальным кодом.
- ✅ **Вступление в комнату** — команда `/entertheroom <код>` добавляет пользователя в существующую комнату.
- ✅ **Профиль пользователя** — кнопка «👤 Мой профиль» показывает ID, username и список комнат с интерактивными кнопками.
- ✅ **Инлайн-меню** — удобные кнопки для навигации.
- ✅ **Справка** — команда `/help` выводит описание всех доступных команд.
- 🚧 **Управление задачами** — планируется добавление задач, расписаний и напоминаний.

---

## 🛠 Технологии

- **Java 25** (совместимость с 21+)
- **Maven** — сборка проекта и управление зависимостями
- **Telegram Bots API** (библиотека `telegrambots`)
- **SQLite** — хранение данных (пользователи, комнаты, участники)
- **VS Code** — среда разработки

## 📦 Установка и запуск

### Предварительные требования
- Установленная Java (версия 21 или выше)
- Установленный Maven
- Telegram-аккаунт и токен бота (получить у [@BotFather](https://t.me/BotFather))

## 🚀 Использование

| Команда | Описание |
|---------|----------|
| `/start` | Показать главное меню с кнопками |
| `/help`  | Вывести справку по командам |
| `/createroom <код>` | Создать комнату с указанным кодом (например, `/createroom myflat`) |
| `/entertheroom <код>` | Присоединиться к существующей комнате |

После создания или входа в комнату она появится в профиле пользователя. Нажатие на кнопку комнаты покажет информацию о ней (в будущем — задачи).

## 🗄 Структура базы данных

### Таблица `users`
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER PRIMARY KEY | Внутренний ID |
| `user_id` | INTEGER UNIQUE | Telegram ID пользователя |
| `username` | TEXT | Имя пользователя (username) |
| `registered_at` | TIMESTAMP | Дата регистрации |

### Таблица `rooms`
| Поле | Тип | Описание |
|------|-----|----------|
| `id` | INTEGER PRIMARY KEY | Внутренний ID комнаты |
| `room_code` | TEXT UNIQUE | Уникальный код комнаты |
| `created_by` | INTEGER | Telegram ID создателя |
| `created_at` | TIMESTAMP | Дата создания |

### Таблица `room_members`
| Поле | Тип | Описание |
|------|-----|----------|
| `room_id` | INTEGER | ID комнаты (внешний ключ) |
| `user_id` | INTEGER | Telegram ID пользователя (внешний ключ) |
| `joined_at` | TIMESTAMP | Дата вступления |
